# Step 3 – Organizational Unit Creation

## Objective

Create an organized Active Directory structure to manage users and resources.

---

## OUs Created

The following Organizational Units were created inside the domain:

lab.local
├── IT
├── Finance
├── Sales
└── HR

---

## Purpose

Each OU represents a department within the organization and allows for:

- Logical separation of users  
- Easier group policy application  
- Department-based permissions  

---

## Tools Used

- Active Directory Users and Computers (ADUC)

---

## Result

A clean and scalable Active Directory structure prepared for user and group management.
