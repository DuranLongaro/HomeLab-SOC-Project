# Step 5 – Joining Client Machines to the Domain

## Objective

Connect a Windows client machine to the lab.local domain.

---

## Steps Performed

1. Installed Windows 10/11 on a client machine  
2. Opened System Settings  
3. Selected **Rename this PC → Join Domain**  
4. Entered domain name: lab.local
5. Authenticated using domain administrator credentials  
6. Rebooted the machine  

---

## Verification

- Logged in using a domain account  
- Confirmed computer appeared in AD under Computers  
- Verified domain authentication was successful  

---

## Result

A working domain-joined client ready for testing permissions and policies.
