# Creating Wazuh Configuration File

## Purpose

The installation requires a configuration file that defines:

- Node names  
- IP addresses  
- Roles in the cluster  

---

## Config File Location

Created on:

wazuh-collector

Path:

/home/meatball/config.yml

---

## Example Configuration Structure

The configuration file defined:

- Indexer node = wazuh-collector  
- Dashboard node = wazuh-siem  
- IP mappings for both systems  

This file was used as the blueprint for certificate generation.
