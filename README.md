## GOAL

Build a Security Operations Center (SOC) style home lab to simulate enterprise security monitoring, log collection, and incident response.

## Overview

This lab is designed to simulate a small enterprise network using:

- Active Directory Domain Controller
- Windows endpoints
- SIEM platform
- Log collection
- Threat detection
- Monitoring dashboards

## Lab Components

- Windows Server running:
  - Active Directory
  - DNS
  - Group Policy

- Additional virtual machines:
  - Windows 10 client
  - Linux server
  - SIEM platform (Wazuh / Elastic / Splunk)
